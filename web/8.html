하가네

상징무기: 망치 /鎚(추)

  • 파괴력 - A
  • 스피드 - A
  • 지속력 - C
  • 정밀조작성 - C
  • 사정거리 - C
  • 성장성 - E

능력 : 원심

내용 :

    -이번 턴의 시작 지점에서 두 칸 이상 뒤로갈 것.
    -이번 턴 동안 아직 공격을 사용하지 않았을 것.

이 조건 하에 발동하는 능력. '행동'이라면 원심이라도 여러 번 사용이 가능.

한 방의 감동이 있는 여신.

공격 카드 '원심격' 하나에 매우 의존하기에 꽤 대처하기 편해 보이지만,
깔봤다가는 종을 울리며 전신에 분쇄골절을 입게 된다. 덤으로 유틸 기술도 우수.
홀수 재구성을 하면 언제나 덱 맨 아래에 원심격이 있는 괴현상에 대해 주의할 것.

아저씨, 들려? '황금향'은 있었어...!




하가네-A (대장장이)

상징무기: 모루 /金床(금상, 모루)

  • 파괴력 - E
  • 스피드 - A
  • 지속력 - C
  • 정밀조작성 - C
  • 사정거리 - C
  • 성장성 - D

능력 : 원심

내용 :

    -이번 턴의 시작 지점에서 두 칸 이상 뒤로갈 것.
    -이번 턴 동안 아직 공격을 사용하지 않았을 것.

이 조건 하에 발동하는 능력. '행동'이라면 원심이라도 여러 번 사용이 가능.

모든 공격 능력을 버리고 다른 여신을 서포트하는 데에 매진하게 된 하가네의 모습.

아무 공격카드나 원심격 급으로 만들어 주기에 파트너 여신의 공격패를 무럭무럭 키우는 재미가 있다.
단점은 느리다는 것.
