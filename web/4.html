토코요

상징무기: 부채

  • 파괴력 - C
  • 스피드 - B
  • 지속력 - A
  • 정밀조작성 - B
  • 사정거리 - D (3~4 미터 정도)
  • 성장성 - B ('어나더'를 '둘' 보유하고 있다)

능력 : 경지

내용 : 집중력이 2일 때 발동한다.

상대의 공격을 흘려보내고 트루 딜을 꽂는 식의 근거리 컨트롤 계 여신.

파워 자체가 강한 편은 아니기에 제 때 알맞은 패를 계획대로 꽂아 넣어야 승리할 수 있다. 집중력의 관리도 중요.




토코요-A

상징무기: 피리

  • 파괴력 - D
  • 스피드 - B
  • 지속력 - A
  • 정밀조작성 - B
  • 사정거리 - C
  • 성장성 - B

능력 : 경지

내용 : 집중력이 2일 때 발동한다.

음악인이 된 토코요.

사거리는 늘어났고 지속력은 조금 좋아졌지만, 추가된 카드의 성능도 그렇고 그나마 있던 공격형 비장패가 사라지는 등 여러모로 미묘. 하지만 패치 후 생각보다 쓸만한 중거리 여신이 되었다.




토코요-A2 (허무신)

상징무기: 공포

  • 파괴력 - C
  • 스피드 - B
  • 지속력 - A
  • 정밀조작성 - A
  • 사정거리 - A
  • 성장성 - B

능력 : 경지

내용 : 집중력이 2일 때 발동한다.

머리띠 색이 바뀐 토코요.

오라 6이상을 유지하는 것을 조건으로 누가 어디에 있든 매 턴 1라이프를 깎아 먹는 원거리 자동 추적형 능력을 갖게 되었다.

하지만 그 능력은 잘 쓰이진 않고, 여러모로 다루기 까다롭지만 특정 조합에서는 무서운 힘을 발휘한다.
