유리나

상징무기: 도

  • 파괴력 - A
  • 스피드 - C
  • 지속력 - B
  • 정밀조작성 - D
  • 사정거리 - D (3~4 미터 정도)
  • 성장성 - B ('어나더'를 '둘' 보유하고 있다)

능력 : 결사

내용 : 라이프가 3이하일 때 강해진다.

후루요니의 가장 기본이 되는 근거리 파워형 여신.

공방일체의 밸런스. 괜찮은 유틸을 가지고 있으며, '결사' 가 켜진 턴의 폭발력은 막강하다.
어쩐지 자주 보이는 기본 여신이라고 만만히 보다간 두개골이 함몰되는 모습을 볼 수 있다.




유리나-A

상징무기: 고도

  • 파괴력 - A
  • 스피드 - C
  • 지속력 - C
  • 정밀조작성 - C
  • 사정거리 - E (2~3 미터 정도)
  • 성장성 - B

능력 : 결사

내용 : 라이프가 3이하일 때 강해진다.

단련 도중이던 시절의 유리나.

사정거리는 짧아지고 전체적으로 더 불안정해졌지만,
'결사' 턴의 그 폭발력은 원본보다도 강하다.




유리나-A2 (조정자)

상징무기: 마음

  • 파괴력 - B
  • 스피드 - C
  • 지속력 - B
  • 정밀조작성 - B
  • 사정거리 - C
  • 성장성 - B

능력 : 결사

내용 : 라이프가 3이하일 때 강해진다.

뭔가 대단한 뭐시기가 된 유리나.

비장패 '월영락'을 잃어 파괴력이 급감했으나 길어진 사정거리와 더불어 상대를 괴롭히는 컨트롤적 능력이 크게 향상되었다.
