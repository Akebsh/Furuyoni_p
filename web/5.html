오보로

상징무기: 인술

  • 파괴력 - B
  • 스피드 - B
  • 지속력 - D
  • 정밀조작성 - C
  • 사정거리 - D
  • 성장성 - C

능력 : 설치

내용 : '설치' 키워드가 적혀 있는 카드는, 덱을 재구성하기 직전 타이밍에 '덮음패' 에서 '한 장' 발동이 가능하다.
(사용 자체는 손패에서 쓰듯이 똑같이 조건을 맞춰 사용한다)

매우 효율적인 능력을 가진 여신.

설치 능력을 이용하면 덮음패도 일종의 손패처럼 생각할 수 있다. 즉 설치를 가진 카드는 마구 덮어도 손해가 별로 없으니 초반 행동의 제약이 덜한 편. 하지만 기초 체급의 문제로 요즘은 조금 힘들다.

덮음패에서 설치로 철사(2/2)를 사용한 후 덱 재구성 - 셔플 - 2드로우 로 다시 철사를 뽑아 사용하는 '철재철'은 오보로 유저라면 누구나 할 수 있어야 한다고 전해진다.

능력 : 설치

내용 : '설치' 키워드가 적혀 있는 카드는, 덱을 재구성하기 직전 타이밍에 '덮음패' 에서 '한 장' 발동이 가능하다.

산전수전을 다 겪은 오보로.

사정거리는 살짝 늘었지만 난이도도 조금 상승했다. '수리검'을 이용한 짤짤이와 신대의 가지, 일명 '얼방'을 이용해 '니가 와' 드러눕기를 잘해야 하는 수비적 여신.




오보로-A2

상징무기: 전자

  • 파괴력 - B
  • 스피드 - B
  • 지속력 - D
  • 정밀조작성 - A
  • 사정거리 - C
  • 성장성 - C

능력 : 설치, 디지 설치

내용 : '설치' 키워드가 적혀 있는 카드는, 덱을 재구성하기 직전 타이밍에 '덮음패' 에서 '한 장' 발동이 가능하다.
(사용 자체는 손패에서 쓰듯이 똑같이 조건을 맞춰 사용한다)

내용 : '디지 설치'란 당신이 재구성을 할 수 있는 타이밍에 재구성을 '안 하겠습니다.' 라고 하고 발동할 수 있는 기믹이다.
즉 덱이 평범하게 있을 때 사용하는 것이다. 덱이 없을 때 사용하면 재구성을 못 하기에 2드로우를 날리게 되는 것이다.

발동은 부품을 모아서 한다. 부품은 카드 효과로 모으고, 1개면 1개효과, 2개면 2개효과. 그런데 부품 2개를 썼다면 그 부품 2개의 각각의 2개 효과가 전부 발동하는 식이다.
부품이 여러개라면 발동하고 싶은 만큼만 공개한다.

솔직히, 좃도 쓸데 없이 난잡하기만 한 것 같습니다.
