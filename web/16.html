야츠하

상징무기: 거울

  • 파괴력 - A
  • 스피드 - C
  • 지속력 - D
  • 정밀조작성 - B
  • 사정거리 - D (3~4 미터 정도)
  • 성장성 - A

능력 : 거울상

내용 : 라이프, 오라, 플레어가 상대와 같은 만큼이 거울 수가 된다. (최대 3)

하이 리스크 하이 리턴의 중거리 파워형 여신.

어마어마한 통상패들의 화력. 대신 강한 패널티를 가지고 있다.
신나서 패다가 상대에게 플레어를 듬뿍 주어 역전 당하는 패턴이 가장 흔하니 주의.

생각보다 거울 수에 너무 집착할 필요가 없다.




야츠하-A (완전태)

상징무기: 꽃

  • 파괴력 - A
  • 스피드 - D
  • 지속력 - C
  • 정밀조작성 - A
  • 사정거리 - C
  • 성장성 - A

능력 : 거울상

내용 : 라이프, 오라, 플레어가 상대와 같은 만큼이 거울 수가 된다. (최대 3)

폭주한 느낌의 야츠하.

비장패는 통상패를 완전태로 바꾸는 '나무' 하나 뿐.
그러나 '심연의 입구'라는 말도 안 되는 카드로 상대의 대가리를 부숴 버리는 무서운 여신.

원래는 '허무로 물들이는 꽃'을 이용해 압박 해 가며 죽이는 플랜도 가능했으나 너프 후 다리를 절단 당하여 지금은 휠체어 신세다.




야츠하-AA (자아)

상징무기: 영혼

  • 파괴력 - A
  • 스피드 - C
  • 지속력 - C
  • 정밀조작성 - A
  • 사정거리 - C
  • 성장성 - A

능력 : 거울상

내용 : 라이프, 오라, 플레어가 상대와 같은 만큼이 거울 수가 된다. (최대 3)

진정한 자아를 찾은 느낌의 야츠하.

원본 야츠하 보다 소소한 아드 벌이를 조금 더 잘한다. 또한 대응불가에 대응할 수 있다는 독보적 특징이 있다.

하지만 그 특징이 강하게 먹히지 않는 메타에서는 미묘한 성능을 자랑한다.
